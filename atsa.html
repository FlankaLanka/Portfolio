ATSA 1234

Role: Solo Unity Developer

ATSA Exam Simulator


ATSA1234 is a prep tool I developed after scoring Best Qualified (BQ), the highest scoring category on the ATSA exam. This tool offers practice sessions on all 7 sections of the exam. These include a differences memory game, variables memory game, spatial recognition game, radar simulation, reading comprehension, logical reasoning, and personality test.

ATSA1234 is the closest match to the real test. Many of the sections also have stat tools that are more in depth than any other ATSA prep. This tool has a welcoming UI to make studying feel less intimidating and more engaging. ATSA1234 is currently free for all Spring 2025 test takers.

An official video guide for the ATSA exam and for this tool will be coming soon when downloadable builds of the project fully releases.
